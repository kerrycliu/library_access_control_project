Sign Up Register for new accounts.

1. Click 'Sign Up' button

2. Enter Username, must be unique.

3. Enter password

4. Select a Role (user, librarian, admin, or guest)

5. Press 'Sign Up' Button

This will redirect user to home page.

Edit User Profile. See for yourself.

1. Successfully login to your account

2. Click the 'Edit Profile' button at the top of the screen

3. Enter in the desired changes into the username and password text boxes

4. Click Save

5. The changes for the username can be seen in the home page adn changes for password can be seen when you login in again

Delete Accounts Admins only

1. Admin must log in.

2. Click 'Delete Account' button

3. Enter in the username of the person to be deleted.

4. Click "Delete".

Add Books Librarians only

1. Librarians must log in.

2. Click "Add Book" button.

3. Enter the title for the book

4. Enter the genere of the book.

5. Click "Add".

Check Out Books Users only

1. Users must log in

2. Click on "Check out" button

3. Book list will be displayed.

4. Enter your username and book title to checkout.

5. Click "Check Out".

User should receive the due date of the book.
